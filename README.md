# 👑 什么是 550W 智能助手 

  >550W 读作 “五五零W” 或是 “小五”。
  >550W 智能助手的主要特征包括：
  >
  >1. 在消费级廉价电子硬件上实现智能化个人和助手。基础AI功能可离线使用（ASR、TTS、NLP等）。
  >
  >2. 严格的个人隐私保护，尽可能少且谨慎的使用云服务。有严格的插件和应用权限管理。
  >
  >3. 功能模块化、可定制化。
  >
  >4. 以一套基于websocket的加密通讯协议（AI00）为核心，实现可跨平台、跨编程语言轻松开发。
  >
  >5. 从实际需求出发，真正帮助人的工作学习。
  >
  >6. 以打造艺术品的心态，打造各个细节。
